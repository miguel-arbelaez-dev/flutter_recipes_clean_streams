Flutter Recipes Clean Streams

Flutter Recipes Clean Streams es una aplicación de recetas completa y profesional, desarrollada con Arquitectura Limpia (Clean Architecture) y patrón MVVM, utilizando Streams para una interfaz reactiva y escalable.

Al abrir la aplicación, se cargan automáticamente recetas aleatorias que puedes explorar. También incluye un buscador de recetas, que permite consultar cualquier receta disponible en la API de Spoonacular.

Dentro de cada receta, el usuario puede acceder a información detallada:

Arquitectura escalable y modular (Clean Architecture + MVVM)

Notas Técnicas

API: Spoonacular (requiere apiKey)

Patrón de arquitectura: MVVM con Clean Architecture

Interfaz: reactiva con Streams

Versión de Flutter utilizada: 3.8.6
