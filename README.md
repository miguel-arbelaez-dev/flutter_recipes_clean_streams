Flutter Recipes Clean Streams

Flutter Recipes Clean Streams es un proyecto completo de aplicación de recetas, desarrollado con Arquitectura Limpia (Clean Architecture) y patrón MVVM, que combina funcionalidades avanzadas y una interfaz reactiva con Streams.

Al abrir la aplicación, se cargan automáticamente recetas aleatorias que el usuario puede explorar. Además, incluye un buscador de recetas que permite consultar cualquier receta disponible en la API de Spoonacular.

Al entrar en una receta específica, el usuario puede ver información detallada, incluyendo:

Ingredientes

Preparación e instrucciones paso a paso

Información nutricional y maridaje de vinos

Todo se presenta de forma clara y organizada, haciendo que la experiencia sea completa y profesional.

Este proyecto demuestra buenas prácticas en Flutter, manejo de APIs, diseño escalable y arquitectura limpia, siendo ideal como referencia profesional para mostrar tus habilidades en desarrollo de aplicaciones móviles.
